

## App Description

## Installation

```bash
$ yarn install
```

## Running the app

```bash
#run migrations
$ yarn migration:run

# development
$ yarn run start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod
```

## Migration


```bash
#migrate
$ yarn migrate --name example
```

## Test

```bash
# unit tests
$ yarn run test

# e2e tests
$ yarn run test:e2e

# test coverage
$ yarn run test:cov
```
